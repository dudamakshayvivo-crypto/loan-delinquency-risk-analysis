# Loan Delinquency Risk Analysis

## Project Overview
End-to-end credit risk analysis project using SQL, Python, Machine Learning and Power BI.

## Tech Stack
- Python
- SQL
- Power BI
- Random Forest
- Pandas, Matplotlib, Seaborn

## Project Components
- Data cleaning with SQL
- Exploratory Data Analysis
- Delinquency risk segmentation
- Feature importance modeling
- Power BI interactive dashboard

## Key Insights
- Delinquency Rate: 16%
- Credit Utilization is major risk driver
- Employment and missed payments affect default risk
